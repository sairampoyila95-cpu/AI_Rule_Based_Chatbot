# 🤖 AI Rule-Based Chatbot

## 📌 Project Overview

This project is a Rule-Based AI Chatbot developed using Python as part of the DecodeLabs Artificial Intelligence Internship Program.

The chatbot interacts with users through predefined rules and responds to different commands using if-elif-else conditions. It continuously runs until the user enters an exit command.

---

## 🎯 Objectives

- Create a simple AI chatbot
- Implement rule-based responses
- Use conditional statements and loops
- Handle greetings and exit commands
- Demonstrate basic Artificial Intelligence concepts
